* QuickCalc consists total 6 functionalities:

- Addition
- Subtraction
- Multiplication
- Division
- Square
- Comparison

* Each of these function is divided into different components (Procedures)
* When a program starts, a menu of instructions appears on the screen guiding the user to press the relevant key for specific operation
* Inside Main PROC, it compares the user input and jumps to specific operation's Procedure
* It works in an infinte loop until, user press '0' to exit.

* It is also able to handle exceptions. For example:
- Throws an error if Divisior is 0

* It takes the input values from 0-9 range.

* It prints a message a 'Thank you' message if user exits the program using Graphics.

NOTE: Output could be of 2 digits depending upon the input values.
